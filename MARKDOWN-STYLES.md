# Markdown стили (для контента)

Дополнительные классы из шаблона `итоговая-верстка-для-сайта`, которые можно использовать в Markdown (через HTML-вставки) или в кастомной разметке:

## Типографика
- Оберни материал в контейнер `.md-article`, чтобы применить типографику: крупные `h1/h2/h3`, параграфы `p`, ссылки `a`, цитаты `blockquote`, списки без стандартных маркеров.

## Маркер-подсветка текста
- Используй `<span class="md-marker" style="--marker-color: rgba(250,148,80,0.4)">текст</span>` для подсветки как маркером.

## Callout-блоки
- Каркас:
```html
<div class="md-callout md-callout--info">
  <div class="md-callout__icon"></div>
  <div class="md-callout__body">
    <h4 class="md-callout__title">Заголовок</h4>
    <div class="md-callout__text">Текст блока</div>
  </div>
</div>
```
- Варианты: `md-callout--info`, `md-callout--idea`, `md-callout--warning` (эмодзи/иконка меняются).

## Списки и цитаты
- Внутри `.md-article` списки (`ul/li`) рендерятся без стандартных bullets, с кастомной точкой. Цитаты (`blockquote`) крупнее и центрируются.

## Примечания
- Цвета и кегли приведены к нашей палитре (`#262728` текст, ссылки `#0066cc`), фон callout — светлый, с мягкой тенью.
- Если нужны другие маркеры/иконки — можно добавить свои модификаторы по аналогии.
